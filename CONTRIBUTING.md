# Contributing to RDR2 Photo Converter

We welcome contributions! Please follow these guidelines to ensure a smooth process.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test thoroughly
5. Commit your changes (`git commit -m 'Add some feature'`)
6. Push to the branch (`git push origin feature/your-feature`)
7. Open a Pull Request

## Bug Reports

Please use the GitHub issue tracker to report bugs. Include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- System information (OS, .NET version)

## Feature Requests

We appreciate feature suggestions! Please create an issue with:
- Clear description of the feature
- Why it would be useful
- Any implementation ideas

## Code Style

- Follow C# naming conventions
- Use meaningful variable and method names
- Add comments for complex logic
- Ensure code is readable and maintainable

## Testing

Please test your changes on multiple systems if possible, especially Windows 10/11.
